# Filmvisarna

## start

to build the react the first time use

```js
npm run init
```

to build it if you have already installed all the dependeces then you use

```js
npm run dev
```
to start the backend then use :

```js
npm run server
```

or if you want a watch flag added: 

```js
npm run serverW
```
